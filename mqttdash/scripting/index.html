MQTT Dash scripting manual (JavaScript)

Text metric ⇒ On Receive handler called when payload received for this particular metric. You have access to the following from the script:

app.publish(topic, payload, retained, qos); - Publish to payload to topic with specified retained flag and qos.

metric.data - Custom runtime data. You can get/set any data, which you need to be preserved between calls. The data shared between On Receive, On Display, On Tap. So you can set data in On Receive and use this data later in On Display.

metric.payload - Received raw payload string. You can modify/transform it before it'll be processed and displayed.

metric.topic - Topic, to which payload was received.
